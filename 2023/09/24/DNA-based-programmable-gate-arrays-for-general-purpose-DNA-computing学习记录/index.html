4.Scalable DPGA architecture

1

文章构建DPGA使用DNA-UTS（DNA-uniform transmission signal）当作通用的传递信号

同时构建了一个molecular instruction set指令集定义所有合法的布线

Twenty-four addressable logic gates of four types (AND, OR, NOT and XOR) were designed as computing units

为了防止inter-DPGA crosstalk,设计了DNA origami register to direct asynchronous computing processing of cascade DPGAs

所有的门的输入输出端口都是通用的，便于扩展

所有门是 dual-rail gate,同时允许两个DNA链的输入输出

整个门电路的运算过程如下:the input of DNA strands triggered strand displacement reactions (SDRs)30 to realize logic functions (Extended Data Fig. 4)

文章通过分析优化发现一个结构可以减少泄露和提高速度

2测试单个门电路

测试结果:the dual-rail results of the AND gate under all four input combinations fell within the correct ON and OFF state regions

在没有任何纯化的条件下电路也表现出一个很好的结果，展现了电路的鲁棒性

3使用DNA–UTS集成门电路

有三种类型的布线:input port-to-gate, gate-to-gate and gate-to-output port

这些布线可以通过分子指令来实现:

  • type 1 instruction (WIR1) conducts an input to a gate
  • type 2 instruction (WIR2) conducts an output signal from an upstream gate to an input port of a downstream gate;
  • type 3 instruction (WIR3) conducts an output signal from a gate to an output port of DPGAs
  • For inter-DPGA wiring we adopted a fourth type of wiring instructions (WIR4s) to realize inter-DPGA DNA–UTS transmission—that is, output signals from upstream DPGA1 were transmitted to downstream DPGA2.

我们使用了noise thresholds and signal amplifiers 来减少布线指令在反应过程中的信号衰减

WIR1,WIR2,WIR3使用DNA链构成，使用了SDR。

通过测试发现前三个指令之间的配合良好，Low- and high-output signals for all input combinations went to the correct states

WIR4通过使用DNA origami register来实现DPGA间信号的传递:In a typical WIR4 the DNA origami register temporally stores the outputs from DPGA1, then a downstream WIR2 transmits the output to the entry gate of DPGA2, providing the directionality for asynchronous execution of cascaded DPGAs

同时发现:the use of DNA origami registers for physical separation of DPGAs restricts transient binding between unmatched DNA strands

5.Multitask reconfiguration

我们实现了一个编译器，可以将自然语言或数学逻辑表达式编译为布线指令。

实现构建了一个乘法电路测试，发现输出的高低信号的对比度很高

为了评估电路执行多重任务的通用性和鲁棒性，implemented various basic circuit structures including single gates, cascaded gates, fan-in and fan-out of gates and complex functional circuits.,最后得到了一个很好的结果

为了测试电路的无差错计算能力，又设计了实验，结果表明了电路又有很高的噪声容忍度

6.Parallel and serial integration of DPGAs

为了实验是否可以将DPGA集成到更大规模的电路，以及它的并行能力，设计了一个平方根计算电路和二次方程的求解电路，得到了一个很好的结果（suggesting that the scalability of DPGA is intrinsic and also independent of the solution transfer methodology. ）

7.DPGA-based nonlinear classifier

为了测试电路在医学方面的应用，使用了DPGA电路和模数转换器（analog-to-digital converters (ADCs)）构造了一个非线性的分类器对于疾病的标志物进行分类，得到了一个好的结果。

8.讨论和总结

  • DPGA circuits can be programmably generated to execute various functions (Extended Data Fig. 2). The programmability of DPGAs is enabled by the use of dual-rail gates featuring uniform input/output interfaces, which allows each gate to be addressed, analogous to that in FPGA
  • DNA origami registers that mediate signal transmission between spatiotemporally separated circuits are the key to DPGA-level integration.
  • The use of DNA origami registers greatly suppresses both the transient binding that limits circuit size and the signal decay that limits circuit depth, thereby facilitating higher-ordered integration of multilayer DPGAs and increasing the programme complexity that can be realized.
